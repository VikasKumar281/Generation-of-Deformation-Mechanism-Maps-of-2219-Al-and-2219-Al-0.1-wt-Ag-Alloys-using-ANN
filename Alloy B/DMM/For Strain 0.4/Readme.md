Strain = 0.4
This folder contains the Power Dissipation Efficiency Map, Instability Map, and Deformation Mechanism Map (DMM) generated for Alloy B(2219 Aluminium Alloy micro-alloyed with 0.1% Ag) at a strain of 0.4.

The maps were developed using ANN-predicted flow stress data and are used to identify safe processing regions, instability zones, and optimum hot working conditions.
