# Strain = 0.2

This folder contains the Power Dissipation Efficiency Map, Instability Map, and Deformation Mechanism Map (DMM) generated for Alloy B (2219 Aluminium Alloy + 0.1 wt% Silver) at a strain of 0.2.

The maps were generated using MATLAB-based DMM analysis and ANN-predicted flow stress data obtained from the selected ANN model (BB_11). These maps are used to identify safe processing regions, instability zones, and optimum hot working conditions during hot deformation.
