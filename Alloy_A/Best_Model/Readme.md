# Best Model - Alloy A

## Selected ANN Model

The ANN model AA_08 was selected as the optimal model for Alloy A (2219 Aluminium Alloy).

### Selection Criteria

- Minimum RMS Error
- Lowest Percentage Deviation
- Highest Prediction Accuracy

### ANN Configuration

- Input Parameters:
  - Strain
  - Strain Rate
  - Temperature

- Output Parameter:
  - Flow Stress

### Network Details

- Feed Forward Neural Network
- Hidden Layer: 9 Neurons
- Activation Function: Logsig
- Output Function: Purelin
- Training Algorithm: Levenberg-Marquardt (trainlm)

### Purpose

The selected model was used for flow stress prediction and deformation mechanism map generation for Alloy A.
